Available Commands

Command Description Usage Action
AWS Create Subnet Create a subnet within an existing VPC. dob aws-subnet --region <region> --vpc-id <vpc-id> --cidr-block <cidr-block> --availability-zone <availability-zone>
Create Target Group Create a target group for a load balancer. dob create-target-group --region <region> --name <name> --vpc-id <vpc-id> --protocol <protocol> --port <port> --target-type <target-type>
AWS Create Listener Create a Listener for a Load Balancer. dob aws-create-listener --region <region> --load-balancer-arn <load-balancer-arn> --protocol <protocol> --port <port> --target-group-arn <target-group-arn> --ssl-certificate-arn <ssl-certificate-arn> --action-type <action-type>
Register Targets Register Targets to a Target Group. dob register-targets --region <region> --target-group-arn <target-group-arn> --targets <target-ids>
Create Load Balancer Create a Load Balancer. dob create-load-balancer --region <region> --name <name> --subnets <subnets> --security-groups <security-groups> --scheme <scheme>
Create Security Group Create a Security Group. dob create-sg --region <region> --vpc-id <vpc-id> --group-name <group-name> --description <description> --inbound-rules <inbound-rules>
AWS Internet Gateway Create an Internet Gateway and attach it to a VPC. dob aws-ig --region <region> --vpc-id <vpc-id> --tags <tags>
Create Elastic IP Create an Elastic IP and optionally associate it with an instance. dob create-elastic-ip --region <region> --domain <domain> --instance-id <instance-id> --tags <tags>
Create Route Table Create a Route Table and add routes to it. dob create-route-table --vpc-id <vpc-id> --region <region> --tags <tags> --routes <routes>
Create EKS Cluster Create an EKS cluster. dob create-eks-cluster --cluster-name <cluster-name> --region <region> --version <version> --role-arn <role-arn> --subnets <subnets> --security-group-ids <security-group-ids>
Create EKS Node Group Create an EKS node group. dob create-eks-nodegroup --cluster-name <cluster-name> --region <region> --nodegroup-name <nodegroup-name> --node-role-arn <node-role-arn> --instance-types <instance-types> --subnets <subnets>
Create S3 Bucket Create an S3 bucket with optional configurations. dob create-s3-bucket --bucket-name <bucket-name> --region <region> --public-access-block --versioning --lifecycle-rules <lifecycle-rules> --logging <logging> --encryption <encryption>
AWS NAT Gateway Create a NAT Gateway in a specific subnet. dob aws-nat-gateway --region <region> --subnet-id <subnet-id> --allocation-id <allocation-id> --tags <tags>
Request SSL Certificate Request an SSL certificate. dob request-ssl-certificate --domain-name <domain-name> --validation-method <validation-method> --subject-alternative-names <subject-alternative-names> --tags <tags>
Run UI Start the UI for Screenplay. dob run-ui --port <port>
Generate Token Generate a token for a user. dob generate-token --username <username>
